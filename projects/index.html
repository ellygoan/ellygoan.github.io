ARCADE MACHINE

Designed and built a custom arcade machine using a Raspberry Pi running RetroPie to emulate classic games.

Engineered the cabinet by repurposing an old dresser and cutting additional wood components to achieve a retro aesthetic and ergonomic design.

Integrated an old computer monitor and speakers, configuring them through RetroPie to ensure seamless audio and video output.

Programmed and configured the Raspberry Pi to support a variety of emulators, optimizing performance and controller mappings for a smooth user experience.

ADVENTURE

Developed a multi-level adventure game in C++ with player movement controlled by arrow keys

Implemented object-oriented design for characters, weapons, armor, and objects

Designed enemy classes (Snakewomen, Bogeymen, Dragons, Goblins) with distinct stats (weapons, armor, dexterity, strength) and behaviors

Programmed varied attack ranges and enemy movement patterns

Built recursive functions to manage game logic and progression through levels

Created borders, obstacles, and level design to challenge players with increasing difficulty

ELECTRIC SKATEBOARD

Designed and built a custom belt-drive electric longboard from scratch

Handcrafted the longboard deck using wood sourced from Home Depot

Integrated an ESC (Electronic Speed Controller) and motor

Installed a belt drive kit with motor mount and 9" trucks for stability and performance

Implemented a wireless remote control system for smooth and responsive operation
